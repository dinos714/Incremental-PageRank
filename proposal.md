# 动态网络下的 PageRank 增量算法与扰动分析

<center>张昊鹏 2023010770 计33</center>

## 选题背景

在现代互联网环境下，网页链接关系具有很强的动态性。课本中介绍的 PageRank 算法基于静态图设计，而当网络结构发生微小变动（例如链接的增、删、改）时，重新进行全量迭代计算会造成巨大的计算资源浪费。我希望利用本次 project 探索如何利用矩阵扰动理论来量化网络变动对排名向量的影响，并以此为基础实现更加高效的增量式更新算法。

## 主要内容

本次 project 主要内容设计如下：

1. 文献调研、理论学习与基础环境搭建。
2. 基于 Wikipedia vote network 数据集（或其他可行数据集），完成静态 PageRank 幂法实现。
3. 推导矩阵扰动界限公式，编写扰动模拟脚本。
4. 尝试开发一种基于残差的增量式迭代算法并进行性能调优（尝试引入加速技术）。
5. 设计对比实验：
   1. 改变扰动比例，记录 rank 的变化量。
   2. 对比增量迭代与全量迭代的开销。
   3. ……
6. 整理、分析并可视化实验数据，根据实验结果完成最终的项目总结报告。

## 预期成果

- 推导出网络结构变化量 $\Delta M$ 与排名误差上限 $||\Delta \mathbf{r}||_{max}$ 之间的函数关系。
- 相较于全量迭代收敛速度更快的增量更新算法及其工程实现。
- 一份包含文献综述、算法设计、理论推导、实验结果与分析的项目总结报告。
